Инструкция по билду: В консоли ввести npm run build

Корневая папка состоит состоит из папок: 
dist, в которой назходится готовый билд;
node_modules, модули, созданные при инициализации проекта;
src, в которой находятся файлы приложения. Состоит из:
  assets, в которой находятся файл с шаблоном команда comman.js и папка css с файлом style.css
  test, где распологаются unit-тесты приложения.
